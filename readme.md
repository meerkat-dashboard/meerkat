# Meerkat

A utility to create and share dashboards for icinga2 checks and hosts.

### Sample config
You can use the `-config` flag to specify an alternative config path
```
HTTPAddr = "localhost:8585"

IcingaURL      = ""
IcingaUsername = ""
IcingaPassword = ""
```
